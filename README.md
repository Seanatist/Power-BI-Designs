**Design Standards**
All dashboards in this repo follow a consistent set of principles:

- **Stakeholder-first layout** — KPI cards at the top, supporting visuals below
- **Consistent colour language** — green for positive, amber for neutral, red for risk
- **Interactive slicers** — every report is filterable by key dimensions
- **Clean titles** — no Power BI default labels, every visual is named for its business meaning
